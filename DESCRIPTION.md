# Repository description

Real-data open resilience workflow for converting rainfall, vegetation, agriculture, infrastructure, population, and macroeconomic data into reproducible district-level climate resilience planning priorities.

The pipeline downloads or reads public data, builds a Uganda district-year panel, constructs climate stress, vegetation response, capacity, exposure, uncertainty, and priority layers, compares the proposed macro--micro score against simple and machine-learning baselines, and exports manuscript-ready tables, maps, audit files, and appendix evidence.
