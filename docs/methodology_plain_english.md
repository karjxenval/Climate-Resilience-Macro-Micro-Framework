# Methodology in plain English

The pipeline asks five questions for every district-year:

1. Was rainfall unusually bad for that district?
2. Did vegetation condition decline in the same season?
3. Does the district appear structurally weaker or stronger using available proxies?
4. How many people are exposed to high-stress conditions?
5. How uncertain is the evidence?

It then compares the integrated score against simpler alternatives such as rainfall-only, NDVI-only, capacity-only, equal-weight, PCA, random forest, and gradient boosting baselines. This prevents the final map from being just a repackaged rainfall map.
